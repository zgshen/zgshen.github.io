2017-09-01

定义

REST 的全称是 Resource Representational State Transfer ，资源以某种表现形式进行状态转移
Resource：资源，即数据；
Representational：某种表现形式，比如用JSON，XML，JPEG等；
State Transfer：状态变化。通过HTTP动词实现。
